    vapi metadata metamodel service: get

    Retrieves information about the service element corresponding to service_id.

    The vapi.metadata.metamodel.service_info contains the metamodel information for the operation elements, structure elements and enumeration elements contained in the service element.

    Request:

    HTTP request

    GET https://{server}/rest/com/vmware/vapi/metadata/metamodel/service/id:{service_id}

    Path Parameters

    Name Type Description
    Required
    service_id string Identifier of the service element.
