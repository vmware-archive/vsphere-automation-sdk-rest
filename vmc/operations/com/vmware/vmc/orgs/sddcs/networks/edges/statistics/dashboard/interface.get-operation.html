    vmc orgs sddcs networks edges statistics dashboard interface: get

    Retrieve interface dashboard statistics for a management or compute gateway (NSX Edge).

    Request:

    HTTP request

    GET https://{server}/rest/vmc/api/orgs/{org}/sddcs/{sddc}/networks/4.0/edges/{edge_id}/statistics/dashboard/interface

    ?interval=string

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).
    sddc string Sddc Identifier. (required).
    edge_id string Edge Identifier. (required).

    Request Query Parameters:

    Name Type Description
    bold = required
    interval string Optional. 60 min by default, can be given as 1 - 60 min, oneDay, oneWeek, oneMonth, oneYear. (optional).

    Response:

    HTTP Status Code: 200
