    vmc orgs sddcs networks edges dhcp leaseinfo: get

    Retrieve DHCP leaseinfo of a management or compute gateway (NSX Edge).

    Request:

    HTTP request

    GET https://{server}/rest/vmc/api/orgs/{org}/sddcs/{sddc}/networks/4.0/edges/{edge_id}/dhcp/leaseInfo

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).
    sddc string Sddc Identifier. (required).
    edge_id string Edge Identifier. (required).

    Response:

    HTTP Status Code: 200

    Representation:

    {

            "host_lease_infos_dto": {

                        "ip_address": "string",
                        "mac_address": "string",
                        "next_binding_state": "string",
                        "starts": "string",
                        "tsfp": "string",
                        "tstp": "string",

    Response Type:

    Name Type Description
    bold = required
    value dhcp_leases com.vmware.vmc.model.DhcpLeases.
    value.time_stamp long The timestamp of the DHCP lease. format: int64

    Optional.

    value.host_lease_infos_dto dhcp_lease_info DHCP lease information.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos host_lease_info[] List of DHCP leases.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].mac_address string MAC address of the client.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].ends string End time of the lease.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].abandoned string Time stamp of when IP address was marked as abandoned.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].cltt string Client Last Transaction Time of the lease info.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].client_hostname string Name of the client.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].starts string Start time of the lease.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].binding_state string Lease's binding state.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].hardware_type string The hardware type on which the lease will be used.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].tsfp string Time Sent From Partner of the lease info.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].uid string Uid to identify the DHCP lease.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].next_binding_state string Indicates what state the lease will move to when the current state expires.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].ip_address string IP address of the client.

    Optional.

    value.host_lease_infos_dto.host_lease_info_dtos[].tstp string Time Sent To Partner of the lease info.

    Optional.
