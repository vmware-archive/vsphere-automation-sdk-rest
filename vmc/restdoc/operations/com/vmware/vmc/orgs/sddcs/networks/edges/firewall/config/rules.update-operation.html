    vmc orgs sddcs networks edges firewall config rules: update

    Modify the specified firewall rule for a management or compute gateway (NSX Edge).

    Request:

    HTTP request

    PUT https://{server}/rest/vmc/api/orgs/{org}/sddcs/{sddc}/networks/4.0/edges/{edge_id}/firewall/config/rules/{rule_id}

    Path Parameters

    Name Type Description
    Required
    org string Organization identifier. (required).
    sddc string Sddc Identifier. (required).
    edge_id string Edge Identifier. (required).
    rule_id long Rule Identifier. (required).

    Request Body Parameters:

    Name Type Description
    bold = required
    nsxfirewallrule nsxfirewallrule (required).
    nsxfirewallrule.rule_type string Identifies the type of the rule. internal_high or user.

    Optional.

    nsxfirewallrule.description string Description for the rule

    Optional.

    nsxfirewallrule.rule_id long Identifier for the rule. format: int64

    Optional.

    nsxfirewallrule.match_translated boolean Defines the order of NAT and Firewall pipeline. When false, firewall happens before NAT. Default : false

    Optional.

    nsxfirewallrule.invalid_application boolean Optional.
    nsxfirewallrule.direction string Direction. Possible values in or out. Default is 'any'.

    Optional.

    nsxfirewallrule.statistics firewall_rule_stats Statistics for the rule

    Optional.

    nsxfirewallrule.statistics.timestamp long Timestamp of statistics collection. format: int64

    Optional.

    nsxfirewallrule.statistics.connection_count long Connection count. format: int64

    Optional.

    nsxfirewallrule.statistics.byte_count long Byte count. format: int64

    Optional.

    nsxfirewallrule.statistics.packet_count long Packet count. format: int64

    Optional.

    nsxfirewallrule.name string Name for the rule.

    Optional.

    nsxfirewallrule.invalid_source boolean Optional.
    nsxfirewallrule.logging_enabled boolean Enable logging for the rule.

    Optional.

    nsxfirewallrule.destination address_FW_source_destination List of destinations. Default is any.

    Optional.

    nsxfirewallrule.destination.exclude boolean Exclude the specified source or destination.

    Optional.

    nsxfirewallrule.destination.ip_address string[] List of string. Can specify single IP address, range of IP address, or in CIDR format. Can define multiple.

    Optional.

    nsxfirewallrule.destination.grouping_object_id string[] List of string. Id of cluster, datacenter, distributedPortGroup, legacyPortGroup, VirtualMachine, vApp, resourcePool, logicalSwitch, IPSet, securityGroup. Can define multiple.

    Optional.

    nsxfirewallrule.destination.vnic_group_id string[] List of string. Possible values are vnic-index-[1-9], vse, external or internal. Can define multiple.

    Optional.

    nsxfirewallrule.enabled boolean Enable rule.

    Optional.

    nsxfirewallrule.application application List of applications. Default is any.

    Optional.

    nsxfirewallrule.application.application_id string[] List of string. Id of service or serviceGroup groupingObject. Can define multiple.

    Optional.

    nsxfirewallrule.application.service nsxfirewallservice[] List of protocol and ports. Can define multiple.

    Optional.

    nsxfirewallrule.application.service[].source_port string[] List of source ports.

    Optional.

    nsxfirewallrule.application.service[].protocol string Protocol.

    Optional.

    nsxfirewallrule.application.service[].port string[] List of destination ports.

    Optional.

    nsxfirewallrule.application.service[].icmp_type string IcmpType. Only supported when protocol is icmp. Default is 'any'.

    Optional.

    nsxfirewallrule.source address_FW_source_destination List of sources. Default is any.

    Optional.

    nsxfirewallrule.source.exclude boolean Exclude the specified source or destination.

    Optional.

    nsxfirewallrule.source.ip_address string[] List of string. Can specify single IP address, range of IP address, or in CIDR format. Can define multiple.

    Optional.

    nsxfirewallrule.source.grouping_object_id string[] List of string. Id of cluster, datacenter, distributedPortGroup, legacyPortGroup, VirtualMachine, vApp, resourcePool, logicalSwitch, IPSet, securityGroup. Can define multiple.

    Optional.

    nsxfirewallrule.source.vnic_group_id string[] List of string. Possible values are vnic-index-[1-9], vse, external or internal. Can define multiple.

    Optional.

    nsxfirewallrule.action string Action. Values : accept, deny

    Optional.

    nsxfirewallrule.invalid_destination boolean Optional.
    nsxfirewallrule.rule_tag long Rule tag. Used to specify user-defined ruleId. If not specified NSX Manager will generate ruleId. format: int64

    Optional.
