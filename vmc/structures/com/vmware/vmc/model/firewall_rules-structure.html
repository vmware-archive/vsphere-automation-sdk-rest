    Optional
    firewall_rules nsxfirewallrule[] Ordered list of firewall rules.

    Optional.

    firewall_rules[].rule_type string Identifies the type of the rule. internal_high or user.

    Optional.

    firewall_rules[].description string Description for the rule

    Optional.

    firewall_rules[].rule_id long Identifier for the rule. format: int64

    Optional.

    firewall_rules[].match_translated boolean Defines the order of NAT and Firewall pipeline. When false, firewall happens before NAT. Default : false

    Optional.

    firewall_rules[].invalid_application boolean Optional.
    firewall_rules[].direction string Direction. Possible values in or out. Default is 'any'.

    Optional.

    firewall_rules[].statistics firewall_rule_stats Statistics for the rule

    Optional.

    firewall_rules[].statistics.timestamp long Timestamp of statistics collection. format: int64

    firewall_rules[].statistics.connection_count long Connection count. format: int64

    Optional.

    firewall_rules[].statistics.byte_count long Byte count. format: int64

    Optional.

    firewall_rules[].statistics.packet_count long Packet count. format: int64

    Optional.

    firewall_rules[].name string Name for the rule.

    Optional.

    firewall_rules[].invalid_source boolean Optional.
    firewall_rules[].logging_enabled boolean Enable logging for the rule.

    Optional.

    firewall_rules[].destination address_FW_source_destination List of destinations. Default is any.

    Optional.

    firewall_rules[].destination.exclude boolean Exclude the specified source or destination.

    Optional.

    firewall_rules[].destination.ip_address string[] List of string. Can specify single IP address, range of IP address, or in CIDR format. Can define multiple.

    Optional.

    firewall_rules[].destination.grouping_object_id string[] List of string. Id of cluster, datacenter, distributedPortGroup, legacyPortGroup, VirtualMachine, vApp, resourcePool, logicalSwitch, IPSet, securityGroup. Can define multiple.

    Optional.

    firewall_rules[].destination.vnic_group_id string[] List of string. Possible values are vnic-index-[1-9], vse, external or internal. Can define multiple.

    Optional.

    firewall_rules[].enabled boolean Enable rule.

    Optional.

    firewall_rules[].application application List of applications. Default is any.

    Optional.

    firewall_rules[].application.application_id string[] List of string. Id of service or serviceGroup groupingObject. Can define multiple.

    Optional.

    firewall_rules[].application.service nsxfirewallservice[] List of protocol and ports. Can define multiple.

    Optional.

    firewall_rules[].application.service[].source_port string[] List of source ports.

    firewall_rules[].application.service[].protocol string Protocol.

    Optional.

    firewall_rules[].application.service[].port string[] List of destination ports.

    firewall_rules[].source.vnic_group_id string[] List of string. Possible values are vnic-index-[1-9], vse, external or internal. Can define multiple.

    firewall_rules[].action string Action. Values : accept, deny
